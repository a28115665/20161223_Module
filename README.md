# 20161223_Module
20161223 開發模組

## 主要模組內容  
- `express` : 4.13.1
- `bootstrap-sass` : 3.3.7
- `gulp` : 3.9.1

## 安裝方式
- Install Node.js (https://nodejs.org/en/)
- Direct download (https://github.com/a28115665/20161223_Module.git)

## 使用方式
```sh
# install dependencies
npm install

# run it
node app.js
```
